# useCursorInteraction

**Category:** Cursors | **Type:** hook

## 📦 Installation & Import

```bash
npm install @instincthub/react-ui
```


```tsx
import React from 'react';
import { useCursorInteraction } from '@instincthub/react-ui';

function MyComponent() {
  return (
    <useCursorInteraction
    />
  );
}
```
